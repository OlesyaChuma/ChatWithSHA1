# ChatWithSHA1

Учебный консольный проект на C++, реализующий простую модель чата с регистрацией и авторизацией пользователей.  
Пароли пользователей хранятся в виде SHA-1 хешей, а данные аутентификации размещаются в хеш-таблице.

---

## Описание проекта

Проект демонстрирует применение:

- хеш-функций (SHA-1);
- хеш-таблицы с закрытым хешированием;
- квадратичного пробирования для разрешения коллизий;
- динамического расширения хеш-таблицы;
- принципов инкапсуляции и безопасной работы с памятью.

Логин пользователя используется как **ключ**, а хеш пароля — как **значение**.

## Основные компоненты

### SHA-1
Реализация алгоритма SHA-1 используется для хеширования паролей.  
Пароли **не хранятся в открытом виде**.

### HashTable
Собственная реализация хеш-таблицы:
- метод хеширования — **метод умножения**;
- разрешение коллизий — **квадратичное пробирование**;
- поддержка удаления элементов;
- пересборка таблицы при большом количестве удалённых записей.

### Chat
Класс, предоставляющий интерфейс:
- `reg()` — регистрация пользователя;
- `login()` — проверка логина и пароля.

---

## Пример использования

```cpp
Chat chat;

chat.reg((char*)"user", (char*)"1234", 4);

std::cout << chat.login((char*)"user", (char*)"1234", 4) << std::endl; // 1
std::cout << chat.login((char*)"user", (char*)"1111", 4) << std::endl; // 0
```

## Устойчивость и безопасность
Пароли хранятся только в виде хешей;

Корректная работа с динамической памятью (new[] / delete[]);

Защита от коллизий и переполнения таблицы;

Корректная обработка удалённых элементов;

Отсутствие артефактов сборки в репозитории (.gitignore).

Алгоритм SHA-1 используется в учебных целях. В реальных системах применяются более современные алгоритмы хеширования.
